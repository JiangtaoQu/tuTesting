# 数据

## 增

```mysql
insert into 【表】 (【字段1】,【字段2】) values ("【数据1】","【数据2】");
```

## 删

```mysql
delete from 【表】 where 【数据条件】;
```

- 清空

```mysql
truncate table 【表】;
```

## 改

```mysql
update 【表】 set 【字段1】=【值】,【字段2】=【值】 where 【数据条件】;
```

## 查

```mysql
select 【字段1】,【字段2】 from 【表】 where 【数据条件】;
```
