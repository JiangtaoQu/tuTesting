# Redis

1. NoSQL——不支持 SQL 语句

2. 储存形式多样—— K:V(string;hash;list;set;zset)

3. 原子性——事务是一个不可再分割的工作单元，事务中的操作要么都发生，要么都不发生。*不会发生多线程错乱*

4. 内存型数据库，高性能——常用作缓存
