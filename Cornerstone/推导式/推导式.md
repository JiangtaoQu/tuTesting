# 推导式

## 列表推导式

- 遍历操作

`[【每一个元素或者是和元素相关的操作】 for i in 【可迭代数据类型】]`

- 双 for 遍历

`[【每一个元素或者是和元素相关的操作】 for i in 【可迭代数据类型】 for ii in i]`

![1562993127866](推导式.assets/1562993127866.png)



- 筛选操作

`[【满足条件的元素相关的操作】 for i in 【可迭代数据类型】 if 【元素满足相关的条件】]`
