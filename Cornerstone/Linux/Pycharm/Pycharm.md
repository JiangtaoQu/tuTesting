# Pycharm

1. 右键安装包，点击 Extract Here

2. 解压tar.gz

```bash
tar -xf pycharm-professional-2019.1.2.tar.gz 
```

   

3. 安装

```bash
cd bin/

sh ./pycharm.sh
```
